## If You Whant to Run This Locally Then Follow the Steps
### Clone / fork the Repo 

### Do a npm/yarn install 

### Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
